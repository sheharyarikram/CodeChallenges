## Overview

This repository contains various coding challenges and exercises in Java.

### To run the coding challenges and check pass/fail:
- Checkout from GIT
- Import as Maven project in Eclipse
- Run the `PlayGroundInitializer` class.


### To add code for a new challenge:
- Add a new Java file under the appropriate `main.java.com.playground` package
- Add a testing method in the Java class
- Invoke the testing method in teh appropriate file under the `test.java.com.playground` package
